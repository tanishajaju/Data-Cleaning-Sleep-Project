# Data-Cleaning-Sleep-Project
Contains python files used for data cleaning and merging

The project was done using 4 python:
1. init.py: initalizes data 
2. analyze.py: Reads the csv files, of the data and calculates the metric required.
3. merge.py: Merges the survey data and the calculated metrics
4. reorder.py: Reorders the according to the initial survey ordering and cleand reduntant columns.
